# hcmc-voting-4rum
## Prerequisite
- Php version from 5.5.x to 7.x
- MySQL

Framework to use: CodeIgniter 3

## Config
- $db['default'] in "application/config/database.php"
- $config['base_url'] in "application/config/config.php"
